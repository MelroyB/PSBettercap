## Main Features
* Control multiple [Bettercap](https://github.com/bettercap/bettercap) nodes through REST API 
* Change wifi adapter or channel per node
* Save GPS location with accesspoint
* Generate KML file 
* Generate csv File
* Save and resume sessions
